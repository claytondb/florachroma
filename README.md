# ?? FloraChroma

> Flower color theme assistant - Design beautiful arrangements with color theory

## ? Features

- **Color Picker** - Choose any color or select from popular flower presets
- **Color Harmonies** - 5 harmony types:
  - Complementary
  - Analogous
  - Triadic
  - Split-Complementary
  - Monochromatic
- **Flower Suggestions** - Real flower recommendations matching your palette
- **Arrangement Preview** - Visual preview with vase mockup
- **Save Palettes** - Store favorite color combinations
- **Color Theory Tips** - Learn the 60-30-10 rule and more
