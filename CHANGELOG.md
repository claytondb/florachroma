# Changelog

All notable changes to FloraChroma.

## [1.1.0] - 2026-02-25

### Added
- **PWA Support** - Install as a standalone app on mobile/desktop
- **Manifest & Icons** - Proper app manifest for installability
- **Offline Support** - Works without internet connection

---

## [1.0.0] - 2026-01-27

### Added
- **Initial Release**
- Color picker with HSL-based selection
- 10 preset flower colors (Rose Pink, Red, Orange, Yellow, Purple, Blue, Green, White, Blush, Lavender)
- 5 color harmony algorithms:
  - Complementary
  - Analogous
  - Triadic
  - Split-Complementary
  - Monochromatic
- Real flower suggestions for each color in palette
- Visual arrangement preview with vase mockup
- Save/load palettes to localStorage
- Color theory tips and the 60-30-10 rule
- Responsive design for mobile and desktop
